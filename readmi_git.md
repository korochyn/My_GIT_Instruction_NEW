### Выполнение коммита.
Для выполнения коммита используется команда *git commit*. Для этого в терминале с открытой папкой-репозиторием необходимо написать 
git commit -m <сообщение к коммиту>. Сообщениек коммиту писать ОБЯЗАТЕЛЬНО.
